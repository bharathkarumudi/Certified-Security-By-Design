   constructing and deconstructing HOL Terms using the tools and
   techniques - \LaTeX{}, AcuTeX, emacs and ML.

   \begin{enumerate}
   \item Problem Statement
   \item Relevant Code
   \item Execution Transcripts
   \item Explanation of results
   \end{enumerate}
